Solid Web Components UI

这是一个基于 Vite + TypeScript 构建的 Web Component UI 组件库，专为 SolidJS 项目设计。

🚀 项目状态

✅ 服务运行正常

你可以通过以下方式访问和使用这个组件库：

  • 访问地址：http://localhost:8080
  • NPM 包名：solid-web-components-ui
  • 测试命令：npm run test

📦 安装使用

在你的 SolidJS 项目中安装：

npm install solid-web-components-ui

使用示例：

import { Button } from 'solid-web-components-ui'; function App() { return ( ); }

🔧 开发命令

  • npm run dev - 启动开发服务器
  • npm run build - 构建项目
  • npm run test - 运行测试用例
  • npm run lint - 代码质量检查
  • npm run publish - 构建并发布到 NPM

🧪 测试用例

项目包含 5 个测试用例，验证按钮组件的功能：

  • ✅ 默认按钮渲染
  • ✅ 不同变体按钮渲染
  • ✅ 不同尺寸按钮渲染
  • ✅ 禁用状态按钮渲染
  • ✅ 按钮点击事件

运行测试：npm run test

🐳 Docker 部署

完整的部署步骤：

# 启动服务 docker compose up -d # 访问服务 http://localhost:8080 # 停止服务 docker compose down

📁 项目结构

solid-web-components-ui/ ├── src/ │ ├── components/ # 组件目录 │ │ ├── Button.tsx # 按钮组件 │ │ └── __tests__/ # 测试文件目录 │ └── index.ts # 入口文件 ├── Dockerfile # Docker 构建文件 ├── docker-compose.yml # Docker 部署配置 ├── package.json # 项目配置和依赖 └── README.md # 项目文档

ℹ️ 注意：这是一个组件库项目，主要用于在其他 SolidJS 项目中通过 NPM 安装使用。
